Offerings

@objc(RCOfferings)
public class Offerings : NSObject

This class contains all the offerings configured in RevenueCat dashboard. Offerings let you control which products are shown to users without requiring an app update.

Building paywalls that are dynamic and can react to different product configurations gives you maximum flexibility to make remote updates.
