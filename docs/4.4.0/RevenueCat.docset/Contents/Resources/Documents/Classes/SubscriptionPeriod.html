SubscriptionPeriod

@objc(RCSubscriptionPeriod)
public class SubscriptionPeriod : NSObject
extension SubscriptionPeriod: Encodable

The duration of time between subscription renewals. Use the value and the unit together to determine the subscription period. For example, if the unit is .month, and the value is 3, the subscription period is three months.
