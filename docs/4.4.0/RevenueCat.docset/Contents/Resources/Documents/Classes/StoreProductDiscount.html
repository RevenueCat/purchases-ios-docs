StoreProductDiscount

@objc(RCStoreProductDiscount)
public final class StoreProductDiscount : NSObject, StoreProductDiscountType
extension StoreProductDiscount: Encodable
extension StoreProductDiscount: Identifiable

Type that wraps StoreKit.Product.SubscriptionOffer and SKProductDiscount and provides access to their properties. Information about a subscription offer that you configured in App Store Connect.
