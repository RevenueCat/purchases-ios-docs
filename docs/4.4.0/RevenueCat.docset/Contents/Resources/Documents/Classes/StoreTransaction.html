StoreTransaction

@objc(RCStoreTransaction)
public final class StoreTransaction : NSObject, StoreTransactionType
extension StoreTransaction: Identifiable

Abstract class that provides access to properties of a transaction. StoreTransactions can represent transactions from StoreKit 1, StoreKit 2 or transactions made from other places, like Stripe, Google Play or Amazon Store.
